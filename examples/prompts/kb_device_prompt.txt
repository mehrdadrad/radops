Retrieves raw configuration snippets for network devices (routers/switches) from the knowledge base.

Use this tool to fetch specific config sections like 'interfaces', 'BGP', or 'OSPF'.

Args:
    query (str): The specific configuration topic (e.g., "BGP configuration", "OSPF").
                 IMPORTANT: Do NOT include the device name or location in this string.
    device (str): The target device hostname (e.g., "router1").
    location (str): The physical site code (e.g., "lax01").

Returns:
    str: The raw configuration text. Do not summarize, reformat, or interpret this output.
